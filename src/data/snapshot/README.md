# Snapshot data

Source of truth for public pages when `FROZEN=1`.

Generated by `scripts/freeze-snapshot.ts`. Commit the JSON; never edit by hand.

To refresh:

```bash
FROZEN=0 npx tsx scripts/freeze-snapshot.ts
git add src/data/snapshot/
git commit -m "data(snapshot): refresh frozen-mode dataset"
```

See `docs/solutions/integration-issues/freezing-and-unfreezing-parentbench.md`
for the full freeze/unfreeze procedure.
