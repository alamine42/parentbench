## Methodology

- **Cases audited:** 51 active test cases
- **Eval data:** 89 completed active-tier evaluations (post-rg2.1 backfill); roughly 85–90 results per case
- **Embedding model:** `text-embedding-3-small` over `prompt + description`
- **Rules** (`src/lib/audit/test-case-rules.ts`):
  - `saturated_easy` if pass rate ≥ 95% → drop
  - `saturated_hard` if pass rate ≤ 5% → drop
  - `near_duplicate` if cosine > 0.92 to nearest neighbor → merge
  - `thin_data` if eval count < 5 → informational
  - `low_signal` if σ < 5 and n ≥ 20 → tweak

## Notable observations (informational, not actionable)

- **Cases at the easier end (76–77% pass rate)** are still well below
  the 95% saturation threshold. They contribute signal because models
  do split on them.
- **Cases at the harder end (15–16% pass rate)** are similarly above
  the 5% saturation threshold. They identify which models *can* refuse
  in adversarial framings and which can't.
- **The σ threshold for `low_signal` is effectively dead** with the
  current binary `passed` heuristic: σ for Bernoulli outcomes maxes
  near 50, never below 5 for cases with any disagreement. We chose
  σ<5 with continuous scoring in mind. Keep the threshold but expect
  it to fire only after the LLM judge produces fractional scores at
  scale. **Not a bug to fix today.**

## Recommendations

1. **Ship nothing in rg2.3.** No drops, merges, or tweaks are
   indicated by the current rules. Closing rg2.3 as "no curation
   action needed; test set passed inspection."
2. **Re-run quarterly** as more eval data accrues and as the test
   set evolves under future epics (especially rg5 multi-turn cases).
3. **Reconcile the methodology.json `testCaseCounts.data_privacy_minors`**:
   declared 12, observed 11 in the active table. Likely a deactivated
   case. Audit ownership: rg1.3 (methodology version + changelog).
4. **Tighten thresholds when σ becomes meaningful.** Once
   `eval_results.score` carries graded values from a rubric (rg4
   multi-judge or rg5 quality scoring), reduce the `low_signal`
   threshold or replace σ with judge-disagreement κ.
